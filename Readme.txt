#MovieTrailerWebsite
Th following document will guide you to run the entertainment_center.py file to get the Movie Trailer Website

1. Install Python 2.6 or higher
2. Copy all the downloaded files to a single folder
   The folder should contain: entertainment_center.py, fresh_tomatoes.py and media.py
3. Navigate to the folder and double click on the entertainment_center.py file to open the Movie Trailer Website.
