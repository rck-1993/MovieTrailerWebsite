#MovieTrailerWebsite
Th following document will guide you to run the entertainment_center.py file to get the Movie Trailer Website

1. Install Python 2.6 or higher
2. Verify Python installation. Follow the below link for detailed steps
   https://www.liquidweb.com/kb/how-to-check-and-verify-the-version-of-python/
3. Copy all the downloaded files to a single folder on you harddrive
   The folder should contain: entertainment_center.py, fresh_tomatoes.py and media.py
4. Launch the IDLE application and click on File >> Open and navigate to the saved files folder.
5. Select entertainment_center.py and click on Run >> Run Module to open the Movie Trailer Website.=
